# new-york-city-housing
### Ohio State Data I/O 2026

**Project Overview** 

This project leverages NYC housing datasets to uncover market trends and determine the primary drivers of property value. By employing data analytics and visualization techniques, the goal is to transform raw sales records into actionable insights presented through an interactive dashboard and a video summary.

**Key Objectives**

- Market Analysis: Examine housing price fluctuations across different boroughs and neighborhoods
- Feature Importance: Identify correlations between price and variables such as square footage, building classification, and location
- Visual Storytelling: Synthesize findings into accessible maps, charts, and tables
- Dashboarding: Develop an intuitive dashboard to make the data understandable for non-technical users

**Deliverables**

- An interactive dashboard highlighting major insights
- A concise video presentation summarizing the analytical process and results

**Technical Focus**
- Exploratory Data Analysis (EDA) using Databricks notebooks
- Data preprocessing, including cleaning and converting raw CSVs into analysis-ready formats
- Communicating analytical results through effective data visualization
